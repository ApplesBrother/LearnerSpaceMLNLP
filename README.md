# IITB Insti-Assist: PDF Q&A Chatbot
---

## Setup/Run Instructions

- **Put in your HUGGING FACE TOKEN and GEMINI API TOKEN: ** I have put mine just for the sake of checking. The imports and installs will be completed on their own.
- **Run the first Collab cell: ** The code will ask you to upload PDF files from your side. Please download the PDFs in the directory and upload them to get the desired answers.
- **After running the first cell: ** Check if all the installations are complete whenever the code throws any error.
- **Run the cells one by one:** Run the cells one by one. The last cell will open a chat box, just like any other chatbot, courtesy of Gradio.
